## The one-paragraph thesis

You do not have a traffic problem — you have an **untapped-warm-audience + no-system** problem. ~30k followers and instant WhatsApp replies already exist; they're leaking because there's no machine catching, booking, reminding, and re-engaging. So we build the machine cheap (≈€20/mo of software), point your existing warm channels into it, and only *then* pour paid budget on top. Paid is the accelerator, not the engine.

### Target economics (internal — never shown publicly)
- Cost per **booked consultation**: €5–20
- Cost per **paying client** (CAC): €40–90 → vs LTV €300–500 = **4–10× return**. Healthy.
- Booking → **show** rate: 70%+ (reminders do this)
- Consultation → **sale**: 35–50% (your job; the machine just fills the calendar)
- If CAC ever exceeds ~€120, we cut that channel.

# PHASE 3 — Systemized lead gen for the next 90 days

**Goal: the funnel runs itself; you optimize numbers, not tasks.** Add CRM structure, lead scoring, and the full WhatsApp lifecycle.

### 3.1 CRM / pipeline (lean — Google Sheet or Notion board; GHL only if you scale)
**Pipeline stages:**
1. New Lead (captured)
2. Engaged (replied on WhatsApp)
3. Booked (consult on calendar)
4. Showed
5. Won (paying client)
6. Lost / No-show
7. Client (active) → Retention
8. Referral source

**Tags:** source (IG / GBP / ad / referral), goal (slăbire / energie / masă), language (RO/EN), temperature (hot/warm/cold), no-show, member, ambassador-candidate.

**Lead scoring (auto, simple):**
- +3 booked a consult · +2 replied within 1h · +2 referral lead · +1 engaged with content · −2 no-show once · −5 no-show twice
- Score ≥5 = "hot" → gets a same-day personal WhatsApp nudge (you tap-approve). <2 = drip only.

**Daily dashboard (one screen, auto-updating):** new leads today · booked this week · shows vs no-shows · consults→sales % · reviews count · CAC. If you only look at one number daily: **booked consultations this week.**

### 3.2 WhatsApp lifecycle automation (the full spine)
Every sequence is pre-written; you tap-approve only where it says "founder."

1. **First-response (instant, auto):** greeting + free-scan pitch + Fresha link + 1 qualifying question. *(Phase 1 message above.)*
2. **Booking nudge (if they don't book in 1h):** "Mai e un loc liber [zi]. Îl țin pentru tine? 👉 [link]"
3. **Reminder sequence (auto, Fresha + WhatsApp):** 24h before ("Te aștept mâine la [oră], Strada Sevastopol 24 — confirmă cu 👍"), 2h before ("Ne vedem în curând! Cod intrare/etaj: …").
4. **No-show recovery (auto):** within 1h of a miss — "Te-am ratat azi 😊 Se întâmplă. Reprogramăm? Aici sunt locurile libere 👉 [link]"; +48h if no reply, one more soft nudge, then drip.
5. **Post-consult (auto):** thank-you + **review request (1-tap Google link)** + if not yet a member, value recap + offer.
6. **Undecided follow-up (auto, 3 touches over 7 days):** address the top objection (price/time/"voi reuși?"), a proof testimonial, then a deadline nudge.
7. **Reactivation (auto, monthly):** dormant leads + lapsed members get a "Ne-a fost dor de tine / hai la o re-evaluare gratuită" sequence.

| Sequence | Effort to build | Cost | ROI | Automation |
|----------|-----------------|------|-----|------------|
| All 7 WhatsApp flows in ManyChat | 1 setup day | ~€20/mo | Very high | Fully auto + tap-approve |
